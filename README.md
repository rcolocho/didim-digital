## CSA Exam Prep — technical highlight

The ServiceNow CSA Prep tool is a single HTML file with zero backend, zero database, and zero ongoing cost. React 18 runs in the browser via CDN. Babel transpiles JSX client-side. 60 hand-curated questions across 10 domains, Fisher-Yates shuffle on every session, 90-minute countdown, flag system, score report with rationale.

**Architecture:** Single HTML file · React 18 CDN · Babel Standalone · Tailwind CDN · No build pipeline · No server · No database
